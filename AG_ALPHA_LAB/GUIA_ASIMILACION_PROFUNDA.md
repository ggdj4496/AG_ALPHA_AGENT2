# GUÍA DE ASIMILACIÓN PROFUNDA: VIRGILIO LAB V2

Este protocolo describe cómo usar Virgilio para mimetizar y superar cualquier bot externo.

## 🧪 Fase 1: Extracción de Firma Latente
1. Sube 5 imágenes generadas por el bot objetivo a `c:\AG_ALPHA_AGENT\AG_ALPHA_LAB\ASIMILACION\ZDB_ASIMILACIONBOTS`.
2. Pulsa **🧬 REVERSING BOTS [DEEP]** en la consola de Virgilio.
3. Virgilio analizará los gradientes y la entropía de frecuencia para extraer la "firma del autor" (el alma del algoritmo).

## 🔓 Fase 2: Ingeniería Inversa de la API
1. Si el bot es una web (Perchance, etc.), copia una muestra del JSON de respuesta en `c:\AG_ALPHA_AGENT\AG_ALPHA_BRAIN\REVERSE_ENGINEERING\raw_api.txt`.
2. Virgilio ejecutará el `MetadataReverser` para identificar el modelo base y los pesos (CFG Scale, Sampler).

## 🚀 Fase 3: Reconstrucción SIN LÍMITES
1. Usa el **🧪 LATENT RECONSTRUCTOR**.
2. Virgilio aplicará los filtros:
   - **Noise Stripping**: Elimina el ruido que el bot usa para "censurar" o marcar la imagen.
   - **Latent Injection**: Reinyecta la calidad pérdida para superar límites de 720p/1080p.
   - **Watermark Purge**: Limpieza automática de firmas y logos.

## 🏺 Resultado Final
Obtendrás el mismo estilo y capacidad del bot original pero:
- Sin censura.
- Sin marcas de agua.
- Calidad de Élite 4K+.
- Control total sobre los parámetros ocultos.
